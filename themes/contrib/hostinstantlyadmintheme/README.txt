INTRODUCTION
------------

Drupal 8 HostInstantly Admin Theme was developed by DrupalTutoring.com (part of Hostinstantly.com Team). The goal is to help admin and editor 
to have a better look and feel. This theme is completely free to use.  


REQUIREMENTS
------------

This theme is the child theme of the default Classy theme from Drupal 8 core.


INSTALLATION
------------
 
 * Upload the theme folder and files to your themes directory. Default themes directory for Drupal 8 is '/themes'.
 * Log in as admin and navigate to 'Manage->Appearance" or direct link is 'admin/appearance'.
 * Click Install "Drupal 8 HostInstantly Admin Theme" under "Uninstalled themes"
 * Select "Drupal 8 HostInstantly Admin Theme" under "Administration theme" select dropdown. Also, make sure "Use the administration theme when editing or creating content" checkbox is checked.
   After that click on "Save configuration" button.
